No Code Sharing or Redistribution

Purchased Pine Script code is licensed to your account only. Sharing, redistributing, reselling, or publishing the indicator code is strictly prohibited and will result in immediate access revocation.

One Account Per User

Each user is limited to one account. Creating multiple accounts to abuse promotions, referral codes, or free trials is not allowed and may lead to all accounts being suspended.

4

Indicator Updates & Maintenance

We regularly update our indicators to improve performance and fix bugs. During maintenance windows, some indicators may temporarily be unavailable. Always keep your Pine Script code updated with the latest version.

5

VIP Access & Expiry

VIP access codes are valid for 30 days from activation. Each code can only be used once and is tied to a single account. Expired access can be renewed by purchasing a new subscription or code.

Community Conduct

Be respectful in all community channels. Harassment, spam, unsolicited promotions, or sharing of harmful content will result in an immediate ban from all Gautam Singh AI platforms.
